# AasmGraph

Add-on gem for creating graphs from AASM state machine definitions

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'aasm_graph'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install aasm_graph
